# m3u8-downloader-vibe

This repo contains a quick and dirty HLS Video downloader I made with Gemini 3
Pro, hence the "vibe" in the repo name.

This program is 100% vibe-coded. **Use at your own risk!**

## Usage

In the releases are binaries for Windows and Linux. Just throw them into a
directory and execute them via a terminal. You will also need a version of
[ffmpeg](https://ffmpeg.org/) installed on your system for this tool to work
correctly.
